# forum-app
A public forum app with user registration, posts, bookmarks, and comments using ExpressJS and VueJS.

Technologies used:
  Front-End:
    VueJS
    Vuex
    Vue-Router
    Vuetify
    Axios
  Backend:
    NodeJS
    ExpressJS
    Passport
    JWT
    MongoDB

Checkout project at : https://forum-app-vks.herokuapp.com/
